# Termux Games
Sebuah tools untuk memainkan permainan-permainan di Aplikasi Termux ðŸŽ®. Dengan [20 daftar game](https://github.com/FajarKim/games-termux#daftar-game) terbaik yang dapat dimainkan tanpa menghabiskan ruang penyimpanan.

# Petunjuk Penginstallan
* `$ pkg update -y && pkg upgrade -y`
* `$ pkg install git`
* `$ git clone https://github.com/FajarKim/games-termux`
* `$ cd games-termux`
* `$ bash install.sh`
* `$ bash games.sh` or `$ ./games.sh`
