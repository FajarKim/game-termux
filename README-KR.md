# 설치 지침
* `$ pkg update -y && pkg upgrade -y`
* `$ pkg install git`
* `$ git clone https://github.com/FajarKim/games-termux`
* `$ cd games-termux`
* `$ bash install.sh`
* `$ bash games.sh` 또는 `$ ./games.sh`
